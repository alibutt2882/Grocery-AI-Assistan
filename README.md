# Grocery-AI-Assistan
Project Introduction: Grocery AI Assistant
The Grocery AI Assistant is an intelligent web application designed to help consumers make smarter, safer, and more cost-effective purchasing decisions. Built using Python and the Streamlit framework, this all-in-one digital companion leverages computer vision and data analytics to analyze products in real-time, helping users verify dietary compliance, assess produce quality, and find the best prices.
